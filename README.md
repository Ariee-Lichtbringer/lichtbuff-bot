# Lichtbuff Bot

Bereinigte Version des Discord-Bots für Worldbuffs und Hordenbuffs.

Enthalten:
- Worldbuff-Übersicht
- Worldbuff-Ticker
- !wb
- !wurf
- !hordenbuff
- !rend
- !rendhelfer
- !rendbei
- !rendchar
- !renddel
- automatische Hordenbuff-Erinnerungen und Cleanup

Nicht mehr enthalten:
- LichtLoot Prio-Check
- RaidHelper-Abgleich
- DC-Abgleich für Lootseiten
- LichtLoot API-Sync

Wichtig:
Der Discord-Token wird ausschließlich über die Railway-Variable DISCORD_TOKEN geladen.
