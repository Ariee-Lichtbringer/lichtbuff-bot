# Lichtbuff Bot

Bereinigte Version des Discord-Bots für Worldbuffs und Hordenbuffs.

Neu in dieser Version:
- Beim Start werden die letzten Nachrichten aus dem Worldbuff-Ticker-Channel rückwirkend eingelesen.
- Dadurch gehen Ticker-Daten nach Railway-Deploys nicht direkt verloren.

Enthalten:
- Worldbuff-Übersicht
- Worldbuff-Ticker
- !wb
- !wurf
- !hordenbuff
- !rend
- !rendhelfer
- !rendbei
- !rendchar
- !renddel
- automatische Hordenbuff-Erinnerungen und Cleanup

Nicht mehr enthalten:
- LichtLoot Prio-Check
- RaidHelper-Abgleich
- DC-Abgleich für Lootseiten
- LichtLoot API-Sync
